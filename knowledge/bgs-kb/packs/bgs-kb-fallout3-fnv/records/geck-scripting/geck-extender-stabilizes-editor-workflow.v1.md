# GECK Extender patches editor limitations and stability issues

The GECK Extender page documents a community patch layer that improves the Fallout 3 and New Vegas GECK editors.
Examples include larger scriptlet windows, faster exit, restored warnings/logging, plugin-loading improvements, and active/master-file workflow fixes.

If an editor-only failure appears while the game and xEdit readback are sane, check whether the workflow assumes these extender fixes.
