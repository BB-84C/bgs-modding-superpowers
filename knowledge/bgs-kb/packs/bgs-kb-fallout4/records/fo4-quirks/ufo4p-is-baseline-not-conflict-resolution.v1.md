# UFO4P is a baseline patch, not a replacement for conflict resolution

AFK Mods hosts the active Unofficial Fallout 4 Patch discussion forum and release/WIP thread.
That makes UFO4P a key baseline for many lists, but not an automatic merge layer for every later mod.

If a later mod wins over UFO4P, the later record wins.
Use xEdit readback to decide whether a compatibility patch is needed.
