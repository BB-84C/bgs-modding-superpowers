# MCP: xEdit Read-Only

Planned purpose: provide a supporting integration around the primary public surface exposed by `xedit-cli`.

`xedit-cli` remains the primary public surface for conflict indexing, scoped inspection, and SQLite-backed drilldown.

This MCP should stay focused on later or supporting integration scenarios where an agent benefits from a bridge layered on top of the existing `xedit-cli` contract.
