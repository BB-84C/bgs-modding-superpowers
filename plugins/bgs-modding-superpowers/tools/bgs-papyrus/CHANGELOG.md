# Changelog

## 0.1.0

- Initial `bgs-papyrus` CLI for Papyrus toolchain detection, CK-backed compile, Champollion-backed decompile, JSON envelopes, and Starfield Guard syntax post-processing.
- Supports `skyrimle`, `skyrimse`, `fallout4`, and `starfield`; Fallout 3, Fallout: New Vegas, and Oblivion are out of scope because they do not use Papyrus.
- Validated with 26 tests plus real Starfield compile/decompile E2E against the Chronomark fixture.
